# Serviço de notificação

Esse serviço é um pedaço do ecosistema de micro serviços de Gerenciamento de Inventário/Produtos.
Sendo esse o serviço de notificação 

### Dependências
* Spring (boot)
* Micrometer
* Zipkin
* Eureka Client
* Kafka (Consumer)

É necessário que tenha um server Kafka rodando

### Como rodar
`docker compose up`
